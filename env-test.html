🔧 配置测试

💡 使用说明

  • AI_BASE_URL: 运行时设置 AI 服务基础 URL
  • AI_API_KEY: 运行时设置 API 密钥
  • VITE_CLI_AI_SERVICE_BASE_URL: 构建时设置 AI 服务 URL
  • VITE_CLI_AI_API_KEY: 构建时设置 API 密钥

测试命令：

# 使用环境变量启动
AI_BASE_URL=http://test:9068/v1 AI_API_KEY=test123 npm run dev

# 或创建 .env 文件
echo "AI_BASE_URL=http://test:9068/v1" > .env
npm run dev
